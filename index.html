Welcome to the Embedded code for Car2Car

This is for the controller that controls RF and Bluetooth communication.

Hardware info

We are currently using an OSU Wunderboard (AT90USB128) for processing and a CC1101 RF module for transmitting and receiving. More changes will come with a Bluetooth module addition,
